# My Recipe Testing

## Deskripsi

Ini adalah proyek pengujian untuk aplikasi resep menggunakan React Testing Library.

## Instalasi

1. Clone repositori:
   ```bash
   git clone https://github.com/Arriziqrzyx/test-my-recipe
   cd test-my-recipe
   ```
2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the App:

   ```bash
   npm start
   ```

4. Run tests:

   ```bash
   npm test
   ```
